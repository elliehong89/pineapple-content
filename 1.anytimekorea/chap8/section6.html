II. Patterns, Expressions & Practice

1. Verb base + 을래요?/ㄹ래요? “Would you like to ~? ; Do you want to ~?”

Usage

Use "~을래요/~ㄹ래요?" is equivalent to “Do you want to ~?,” “Will you ~?,” “Would you like to ~?” in English.

meaning

"~을래요/~ㄹ래요?" is equivalent to “Do you want to ~?,” “Will you ~?,” “Would you like to ~?” in English.

Form

Use the rule below to choose between the different forms

▶ Verb base (ending in a consonant) + 을래요? :
먹을래요?
[ㄱ + 을래요]
▶ Verb base (ending in a vowel) + ㄹ래요? :
갈래요?
[ㅏ + ㄹ래요]
* Verb base (ending ㄹ) + 래요? :
만들래요?
[ㄹ + 래요]
1.
A:
라면 먹을래요?
Would you like to have ramen?
B:
좋아요.
Sure.
2.
A:
뭐 마실래요?
What would you like to drink?
B:
콜라요.
Cola.
3.
A:
오늘 같이 영화 볼래요?
Do you want to watch a movie together today?
B:
무슨 영화요?
What movie?
4.
A:
콘서트에 같이래요?
Do you want to go to a concert together?
B:
무슨 콘서트예요?
What concert is it?

When you use "~을래요/~ㄹ래요?" in a statement sentence, you express your desire or inten tion. In this case, "~을래요/~ㄹ래요." is equivalent to “I want to ~,” “I will ~” in English.

1.
A:
뭐 드실래요?
What would you like to have?
B:
저는 김밥 먹을래요.
I’d like to have gimbap.
2.
A:
뭐 드실래요?
What would you like to drink?
B:
콜라 마실래요.
I’d like to drink cola.
3.
A:
한국 영화 볼래요?
Do you want to watch a Korean movie? (Or,) Do you want to watch an American movie?
B:
오늘은 미국 영화 볼래요.
As for today, I want to watch an American movie.
4.
A:
저는 아메리카노 마실래요.
I’d like to have Americano
B:
저는 카푸치노요.
For me, cappuccino.
