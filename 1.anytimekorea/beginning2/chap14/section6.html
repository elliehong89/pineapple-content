II. Patterns, Expressions & Practice

1. Verb or adjective base + ~이면/면, ~으면/면 “if ...” or “when ...”

Usage

Use ~으면/~면 to indicate conditional/hypothetical situations or those that are certain to take place. ~으면/~면 is equivalent to the English “if ” or “when.”

Form1

NOUN + 이면/면 (Lit., If it is NOUN)

▶ NOUN + 이면 (When NOUN ends in a consonant)

▶ NOUN + 면 (When NOUN ends in a vowel ) [이 is dropped]

1.

5월이면 [ㄹ + 이면]

날씨가 따뜻해요.
In May, it is warm. [Lit., If (it) is May, the weather is warm.]
2.
두 시간이면 가요.
In two hours, you can get there. [Lit., If (it) is two hours, (you) go.]
3.

한 시면 [ㅣ + 면]

좋을까요?
Will 1 o’clock be good? [Lit.,, If (it) is 1 o’clock, will (it) be good?]
4.
학생이면 무료예요.
For a student, it is free. [Lit., If (one) is a student, (it) is free.]
5.
다음 주면 방학이에요.
It’s a school break next week. [Lit., when (it) is next week, it is a school break]

useful expressions

무료
no charge; free
방학
school break
Form2

Verb/Adjective + 으면/면

▶ Verb/Adjective + 으면 (When Verb/Adjective ends in a consonant)

▶ Verb/Adjective + 면 When Verb/Adjective ends in a vowel)

* Verb/Adjective base (ending in ㄹ) + 면

1.
내일 날씨가

좋으면 [ㅎ + 으면]

등산 가요.
If the weather is good tomorrow, let’s go mountain climbing.
2.
이번 주말에

바쁘면 [ㅡ + 으면]

다음 주말에 만나요.
If (you) are busy this weekend, let’s meet next weekend.
3.
일이 끝나면 저한테 문자하세요.
When your work is over, please text me.
4.
뉴욕에 오면 저한테 전화하세요.
When you come to New York, please call me.
2.
한국에

살면 [ㄹ + 으면]

한국어를 빨리 배워요.
If you live in Seoul, you learn Korean quickly.

useful expressions

등산 가다
to go hiking
문자하다
to send a text message
